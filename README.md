# VOPO-RODNIK-VOTE-CRYPTO

**VOPO-RODNIK-VOTE-CRYPTO** — телеграм-бот с расширенными возможностями для организации и ведения виртуального крипто-портфеля.

---

## Возможности

- **Ведение крипто-портфеля:**  
  - Добавляйте виртуальные токены и следите за их балансом.  
  - Покупайте и продавайте токены (виртуально) через команды бота.  
  - Просматривайте актуальную стоимость и динамику вашего портфеля.  
- **Команды для пользователей и администраторов:**  
  - **Пользователь:**управляйте портфелем.  
  - **Администратор:** создавайте и удаляйте события, просматривайте статистику  портфелей.  
- **Асинхронная база данных:** Используется **aiosqlite** для быстрого хранения данных и обработки запросов.  
- **Актуальные цены криптовалют:** Интегрировано с внешним API (например, CoinGecko) для получения текущих курсов.

Команды

Для пользователей
/start — Запуск бота и список доступных событий и функций.
/reset — Перезагрузка текущего состояния бота.
/portfolio — Просмотр баланса и состава крипто-портфеля.
/buy <токен> <количество> — Виртуальная покупка криптовалюты.
/sell <токен> <количество> — Виртуальная продажа криптовалюты.

Для администраторов
/add_event — Добавить новое событие (голосование или мастер-класс).
/view_events — Посмотреть и удалить события.
/visualize_votes — Просмотр статистики голосований.
/visualize_workshop — Просмотр статистики мастер-классов.
/view_portfolios — Просмотр статистики крипто-портфелей пользователей.
